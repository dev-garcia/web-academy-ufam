A Linguagem HTML

HTML (significa Linguagem de Marcação de Hipertexto), é uma linguagem de Marcação utilizada na construção de páginas nna Web. Documentos HTML podem ser interpretados por navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tags Associadas Descrição
Links 'a' Criar links entre páginas.
Listas ul e li Criar listas de itens não ordenadas.
ol e li Criar listas ordenadas.
Imagens img Inserir Imagens.
